# Father's Playbook — App Marketing Site

The marketing and landing website for the **Father's Playbook** mobile app, a health resource for expectant and new fathers developed in partnership with UT Health Austin.

## Overview

Father's Playbook is a mobile application designed to support fathers through pregnancy, birth, and early parenthood — providing educational resources, guidance on perinatal health, and connections to healthcare providers. This repository contains the public-facing marketing website with App Store and Google Play download links, feature highlights, an about section, and a provider directory.

## Pages

| Page | Content |
|------|---------|
| `index.html` | Home / hero with app download CTA |
| `features.html` | App feature breakdown |
| `about.html` | Project background and UT Health partnership |
| `providers.html` | Healthcare provider directory |
| `appprivacy.html` | Privacy policy |

## Tech Stack

- **Bootstrap 4** — responsive grid and components
- **Font Awesome 5** — icons
- **Adobe Fonts (Typekit)** — custom typography
- **Vanilla JavaScript / jQuery** — interactive features

## Running Locally

No build step required — open any `.html` file directly in a browser, or use a local server:

```bash
npx serve .
```
